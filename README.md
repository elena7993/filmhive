### 2. 목적

이 앱을 통해 사용자들이 영화를 빠르게 검색하고 영화를 보기 전 영화에 대한 정보를 얻을 수 있다.

### 3. 기능

- 영화 검색
- 영화 상세 페이지
- 영화 트레일러 영상보기

### 4. 스택

- HTML, CSS, JavaScript, React, TMDB API, Figma
